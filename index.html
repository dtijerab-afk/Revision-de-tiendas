Configurar Firebase

Para sincronizar con Firebase:

  1. Ve a Firebase Console
  2. Crea un nuevo proyecto o usa uno existente
  3. Ve a Configuración del proyecto → Aplicaciones web
  4. Copia las credenciales debajo
  5. Configura Firestore en "Crear base de datos"
  6. Reemplaza las reglas con las que se muestran abajo

Reglas de Firestore (Copiar en Firebase Console) 

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /tareas/{document=**} {
      allow read, write: if true;
    }
  }
}

⚠️ Nota: Estas son reglas públicas. Para producción, implementa autenticación.
